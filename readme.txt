Add 2d and 3d maps into your posts. And place posts and medias onto the maps.

== Description ==

[Trail-Passion](http://trail-passion.net) is a web site dedicated to outdoor sports. It provides easy ways to show GPS tracks on flat maps, elevation graphs and event 3D views. 
If you want to show your last hicking or race in your blog, this plug-in is made for you.

Not only can you add maps on wordpress, but also put markers on your maps based on the GeoMashup localization of your posts and medias.


== Installation ==

Just unzip the file **trail-passion.zip** in your `/wp-content/plugins` directory and activate it through the 'Plugins' menu in WordPress.


== Frequently Asked Questions ==

= Do I need to get an account on trail-passion's website to use this plugin? =

No. You can display all the public traces of the trail-passion's website without registration.
Registration is only needed if you want to store maps of your own.

= Did you provide an application to easily track routes with GPS on smartphone? =

Yes! We provide an [Android App](https://play.google.com/store/apps/details?id=com.trailpassion "Google Play") for this.
